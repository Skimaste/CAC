Denne mappe indeholder vores færdige modeller

- final_model er vores neurale netværk
- final_weights er vægtene til netværket
- final_cutoff er cutoff værdien som bruges når det neurale netværk prædikterer 0 og 1.
- final_scaler bruges til at skalere testdataen i forhold til træningsdataen.

- logistic_regression_model er vores logistiske regressionsmodel og de tilhørende vægte.
- logistic_cutoff er cutoff værdien som bruges når den logistiske regressionsmodel prædikterer 0 eller 1.
